社群對我職涯的影響

R 社群

  • 2012年10月，我和家齊決定創辦R 社群，尋找R 的同好
    • 我是數學系、統計所，又會寫程式(R, C++)的新鮮人
    • R 當時是個小眾軟體（但是我運氣很好，它起飛了）

跨領域

  • 我的統計不是頂尖
  • 我寫程式的能力也不是頂尖
  • 但是在R 的世界，我似乎走在很前面...

在R 社群，我不是拿到東西，反而是一直給

  • 參與研討會
  • 辦活動
  • 籌備workshop
  • 寫套件

給的人，收穫更豐碩

  • 舉辦或參加很多活動，認識很多人，就獲得更多的邀請
    • g0v ==> 陳老師 ==> 資料科學愛好者年會
    • R 社群 ==> Data Science Program ==> 智庫驅動 ==> 各種教學的機會
  • 認識更多的人，貢獻更多，就更增廣見聞

給的人，收穫更豐碩

  • 社群的經驗，幫助我取得更好的工作
    • 社群 ==> 證明我是個愛好貢獻的人 ==> 找工作增加了籌碼
    • 給Talk、辦活動 ==> 證明我自己的能力 ==> 找工作增加了籌碼

給的人，收穫更豐碩

  • 社群的交流，讓我定位與調整自己的價值
    • 給Talk，反應不大 ==> 我沒有打到需求 ==> 反省與調整作法
    • 避免敞帚自珍

社群是什麼？

  • 一個大坑
    • 坑需要人跳
    • 坑越跳越大
  • 需要第一個跳的人，而且要跳得好

社群是什麼？

  • 一個放大講者價值的「機會」
    • 讓大家看到你
    • 讓你找到貢獻的機會
  • 一個認識有價值的朋友的「地方」
    • 有價值，就會帶來人氣

同學如何利用社群？

  • 沒社群，就創一個
    • 創立者是最累，但是做起來最好收割
    • Open Source 社群也是很值得經營，這一切取決於你的「志趣」
      • Github

同學如何利用社群？

  • 加入既有社群，打入貢獻圈
    • 不要只想著拿東西，要尋找貢獻的機會，主動出擊

社群會如何回饋同學？

  • 申請獎學金、申請學校、公司
    • 貢獻讓你脫離沒經驗
  • 但是不要為了這樣的目的去做貢獻，而是為了價值

社群會如何回饋同學？

  • 讓人來認證自己價值的好地方
  • 讓世界察覺你的價值的好地方
  • 讓他人認證你的價值的好地方
    • 免費的證照

社群會如何回饋同學？

  • 更多的貢獻機會
    • 如果是志趣，就自然會以這些貢獻為樂、為榮
    • 這些貢獻，就是經驗

社群會如何回饋同學？

  • 認識更多的同好
  • 獲得合作的機會
  • 人脈

社群對我職涯的具體影響

  • 工作
  • 課業
  • 事業
  • 交流

總結

  • 探索自己、尋找自己的興趣
  • 摸索結合興趣與專長的方向
  • 尋找自己的同好
  • 深耕與交流
